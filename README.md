# Hartwall Product Showcase
Landing site showcasing Hartwall's beverage products with an interactive flavor carousel and dynamic product displays.

## Getting Started

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:3000`.

## Tech Stack

- React 18
- TypeScript
- Vite
- SCSS Modules
